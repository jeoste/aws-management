import boto3

# Initialize SQS client
sqs = boto3.client('sqs', region_name='us-east-1')

# Send message to queue
response = sqs.send_message(
  QueueUrl='https://sqs.us-east-1...',
  MessageBody='{"order_id": "89201"}',
  MessageAttributes={
    'Type': {'StringValue': 'Order', 'DataType': 'String'}
  }
)
print(response['MessageId'])

/// MESSAGE LIFECYCLE

01. Ingestion & Topic Routing

Producers publish messages to SNS topics. Subscription filters route payloads to specific queues based on attributes.

02. Queue Buffering

SQS persists messages redundantly across availability zones. Visibility timeouts protect against processing failures.

03. Consumption

Workers poll queues for batches. Successful processing deletes the message; failures are routed to DLQs for analysis.
